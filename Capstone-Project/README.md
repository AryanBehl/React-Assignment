# Movie Discovery & Analytics Dashboard

Author: Aryan Behl (2501350047)
Course: B.Tech CSE 1st Year
University: KR Mangalam University

## Features
- Search movies using OMDb API
- Filter by year
- Sort by year
- Pagination
- Chart (movies per year)
- Dark mode
- Watchlist

## Tech Stack
React + Vite, Redux Toolkit, Axios, Recharts, CSS

## How to Run
npm install
npm run dev
